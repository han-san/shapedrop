This is a tetris clone. (Please don't sue me)

Controls
--------

* Up: Hard drop
* Down: Soft drop
* Left: Shift left
* Right: Shift right
* Z: Rotate Left
* X: Rotate Right
* 1: Decrease window size
* 2: Increase window size
* Space: Hold shape

Dependencies
------------

* [SDL2](https://www.libsdl.org)
* [GSL](https://github.com/microsoft/GSL)
* [{fmt}](https://fmt.dev)
* [OpenGL](https://www.opengl.org)
* [Glad](https://glad.dav1d.de)
